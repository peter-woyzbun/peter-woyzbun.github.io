Papers

Analyzing Interprovincial Migration Determinants in Canada: A Random-Effects Gravity Model Approach

I wrote this research paper with a classmate for a forth year applied econometrics course at Queen’s. In it, we use a random-effects gravity model to analyze the migration of Canadians between provinces. We identified “push”, “pull”, and “frictional” factors that impact migration flows.

Push factors encourage outflows of migration from a province, pull factors encourage inflows to a province, and frictional factors discourage both. Our findings are summarized in the figure below.

Structural Breaks and the Bank of Canada Equation

This paper was written for my forth year "economics seminar" class, where we had to do original research and present our findings. In it, I revisit the so-called "Bank of Canada Equation", a model developed in the early 1990s for describing the behaviour of the CAD/USD exchange rate.

Query Graph

Query Graph is a Python package I've been working on as a side project. It's a framework for joining data across different types of relational databases (PostgreSQL, MySQL, Microsoft SQL, and SQLite are currently supported).

In Query Graph, an SQL query on a given database is represented as a graph node.
